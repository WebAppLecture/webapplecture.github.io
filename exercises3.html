01. Grundgerüst

Erstelle die Datei gears.html mit html, head und body Elementen. Erstelle im body ein div Element und gib ihm die Klasse "gear".

Erstelle die Datei gears.css und erstelle einen Eintrag für die Klasse "gear".

Füge dem head in gears.html eine externe Referenz für gears.css hinzu.

Erweitere den Eintrag in gears.css um folgende Eigenschaften

  • Er soll als inline-block dargestellt werden
  • Höhe und Breite sollen 100px betragen
  • Die Hintergrundfarbe ist #777
  • Der Radius der Border beträgt 50% (Das Element ist rund)

02. Die Div-Box, die hat Zähne

Erweitere dein Stylesheet um den Zahnrad-effekt zu erreichen.

  • Die Border: 10px breit, gestreift(dashed), #bbb
  • Um den Hintergrund nicht hinter der Border zu sehen, setzen wir dessen clip Eingeschaft: background-clip: padding-box;

In Google Chrome sieht dies etwas ungleichmäßig aus, in Firefox verteilen sich die "Zähne" perfekt.

03. Schattenspiel

Um dem Zahnrad Plastizität zu verleihen, nutzen wir verschachtelte box-shadows.

        box-shadow: 0 0 0 8px inset #ccc, 
            0 0 5px 12px inset #777, 
            0 0 5px 20px inset #aaa, 
            0 0 0 25px inset #777, 
            0 0 0px 30px inset #ccc,
            0 0 0px 50px inset #001;

04. Animation

Erstelle nun eine Keyframeanimation @keyframes rotate {...} in deinem Stylesheet.

Animiere die Rotation von 0° bis 360° mit der transform: rotate(deg) Eigenschaft.

Füge dem Eintrag für "gear" eine Animation hinzu: animation: rotate 5s normal linear infinite;

  • rotate: Bezeichnet die Keyframeanimation die verwendet werden soll.
  • 5s: wie schnell soll sich das Zahnrad drehen?
  • normal: das Zahnrad läuft die Animation von vorne nach hinten durch.
  • linear: die Animation wird linear durchlaufen.
  • infinite: die Animation läuft endlos.

05. Mehr Zahnräder

Füge nun in gears.html hinter dem vorhandenen div eine weiteres ein.

Dieses soll die Klassen "gear" und "reverse" besitzen.

Füge in gear.css einen Eintrag für "reverse" ein:

  • die Animation soll rückwärts laufen. animation-direction: ...
  • die Animation soll um 1s verzögert starten. animation-delay: ...
  • Setze margin-left: -7px; um die Zähne zum Eingriff zu bringen.

A whole new kind of Casio

Sieh dir eine mögliche Lösung an.

Erweitere den Taschenrechner von gestern mit deinen ganz eigenen Styles, erfülle hierbei diese Kriterien:

  • Verwende mindestens einen Gradienten.
  • Verwende box-shadows um im Stil des Material Designs "3D" Effekte zu erhalten. (Sieh dir im Beispiel an wie der "klick" effekt auf Buttons bei beliebiger Farbe möglich ist)
  • Ändere die Form und Farbe der Tasten. (border-bottom-right-radius: _px _px, usw erlaubt genaue Kontrolle über Eckradien)
  • Füge eine Beliebige Animation hinzu. (Muss nicht der Taschenrechner selbst sein, siehe Beispiel)

Fahrzeug

Baue ein Kettenfahrzeug aus gestylten div-Boxen, erfülle folgende Anforderungen:

  • Das Fahrzeug soll auf dem Bildschirm hin und her fahren.
  • Die Kette muss animiert sein. Orientiere dich am styling der Zahnräder, für die Animation musst du die Kette in Abschnitte zerlegen.
