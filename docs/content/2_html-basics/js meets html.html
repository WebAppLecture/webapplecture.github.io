JavaScript in HTML nutzen

Bevor Javascript im Browser ausgeführt werden kann muss es im HTML eingebunden werden. Externer Javascript wird ideal im <head> eingebunden.

<script defer src="pfad/zum/beispiel.js"></script>

MERKE: defer sorgt dafür, dass das Laden des Codes das Rendern der Seite nicht blockiert!

Um schnell etwas auszuprobieren kann man Skripte auch direkt einbinden.

HTML auslesen & manipulieren

In Javascript greifen wir über das document Objekt auf die HTML Seite zu.

Gezielter Zugriff auf einzelne HTML Elemente: document.querySelector(selector), oder mehrere Elemente: document.querySelectorAll(selector)

Selector Syntax (entspricht CSS Selector Syntax):

  • querySelector("canvas") um auf <canvas> zuzugreifen.
  • querySelector("#target") um auf <div id="target"> zuzugreifen.
  • querySelector(".target") um auf <div class="target"> zuzugreifen.
  • querySelector("[data-value='33']") um auf <div data-value="33"> zuzugreifen.
  • Es gibt noch unzählige weitere Möglichkeiten.
