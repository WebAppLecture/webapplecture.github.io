Async Await

  • Zusätzlich zu Promise.then gibt es mittlerweile auch eine weitere Schreibweise um Asynchronität darzustellen
  • Funktionen die mit async gekennzeichnet sind können intern andere asynchrone Aufrufe (Promises o.ä.) mit await abwarten
  • Fehler werden mit try/catch Blöcken gefangen, vgl. .catch bei Promises
  • Dadurch ergibt sich ein Syntax der sich wie synchrones Javascript liest
  • Async Await kann beliebig mit Promise.then Syntax gemischt werden
  • Nehmt die Schreibweise her, die euch besser liegt, es gibt keine falsche Antwort
