Responsive Design

  • Mit CSS3 wurde auch ein neuer Ansatz für Webseiten Design eingeführt: sog. Media Queries
  • Dies bedeutet, dass die Webseite auf das verwendete Gerät (z.B. Mobiltelefon, PC, ...) und seine Eigenschaften (z.B. Größe, Auflösung, ...) reagiert
  • Dies geschieht durch das media Attribut von Stylesheets, z.B. 
    <link rel="stylesheet" href="screenklein.css" media="screen and (max-width: 600px)" />
<link rel="stylesheet" href="allesklein.css" media="all and (max-width: 600px)" />
<link rel="stylesheet" href="smartphone.css" media="(max-device-width: 600px)" />
  • Alternativ auch direkt im Stylesheet mittels @media
  • Wie kann dergleichen aussehen? Wir schreiben uns CSS-Dokument und bauen dies z.B. so auf: 
    /*Gemeinsame Regeln */
@media screen /* Nur für Screen */
{
    body { width: 75%; }
}
@media print /* Nur für Print */
{
    body { width: 100%; }
}
  • Die media bzw @media-Regeln können auch sehr speziell und kompliziert aufgebaut werden (AND, OR, ...)
  • Weitere Abrufbare Eigenschaften sind (Auszug): height bzw. device-height, orientation (portrait, landscape), aspect-ratio bzw. device-aspect-ratio, color (Farbtiefe in Bits) und resolution (Dichte der Pixel in dpi)
