Funktionen

  • Diese werden über das Schlüsselwort function erstellt. function f(x) {}
  • Jeder Parameter ist in JavaScript optional - daher können Funktionen nicht überladen werden
  • arguments (Array) gibt Zugriff auf alle übergebenen Parameter
  • this gibt Zugriff auf den Scope in dem die Funktion aufgerufen wurde
  • Man kann Standardwerte für Parameter definieren: function f(x = 5) { }
  • Funktionsaufrufe geschehen über den Funktionsnamen und den Parameterwerten in Klammern f(5)
  • JavaScript kennt keine explizit genannten Rückgabewerte, da die Sprache dynamisch ist
  • Standardmäßig wird undefined zurückgegeben - außer man gibt mit return etwas zurück
  • Funktionen können auch anonym erstellt werden function(x) {}
    • Für einmalige Verwendung, als IIFE (Immediatly-Invoked Funtion Expression): (function(){})()
    • Zum speichern in einer Variable (verhindert Hoisting!): const f = function(){}
    • Als Parameter in einer anderen Funktion f(function(){})
