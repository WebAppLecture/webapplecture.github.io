Variablen deklarieren

Ermöglichen uns Daten mit einem Namen zu versehen um dann über diesen Namen auf sie zuzugreifen.
  • Syntax: let varibalenName = wert
  • Beispiel: let farbe = '#f00'
  • Konvention Schreibweise: wortWortWort ("Camel Case")
Der Wert der Variable kann nach der Definition beliebig oft neu beschrieben werden.
  1. let notiz = "noch einkaufen!"
  2. notiz = "erledigt!"
Für aufeinander folgenden Variablendeklarationen gibt es eine Kurzschreibweise:
  • let rot = "#f00", blau = "#00f" entspricht let rot = "#f00" let blau = "#00f"
Für Variablen deren Werte sich nicht ändern soll, verwendet man const. Hier führt der Versuch die Variable zu überschreiben zu einem Fehler.
  1. const PI = 3.1415
  2. PI = 5 wirft eine Fehlermeldung!
  3. Konvention Schreibweise (nur für echte Konstanten!): WORT_WORT_WORT
MERKE: let statt var!

var ist ein alter Variablenbezeichner, welcher noch in vielen Beispielen und Dokumentiation zu finden, sollte aber durch let ersetzt werden.
