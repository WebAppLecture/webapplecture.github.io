01. Das Grundgerüst

Erstellen Sie das Grundgerüst einer Webseite mit allen (benötigten) Elementen einschließlich eines Absatzes (der Tag dafür ist <p>), der ein wenig Text und ein Bild, das durch den <img>-Tag angezeigt wird, enthält.

02. Es lebe die Struktur

Verwenden Sie die Webseite aus Übungsaufgabe 1. Erstellen Sie anstelle des Textabsatzes auf der Seite einen Artikel, der jeweils einen Kopf- und einen Fußbereich besitzt. Fügen Sie dem Artikel zwei Textabsätze hinzu.

Im ersten Textabsatz sollten noch ein wenig Text mit hervorgehobenen <em> ) Wörtern, kursiven ( <i> ) Ausdrücken und fett ( <strong> ) dargestellten Zahlen geschrieben werden. Im zweiten Absatz sollten Informationen zum Autor ( <address> ) und zur letzten Erstellungszeit ( <time> ) der Seite angezeigt werden.

03. Weitere Funktionalitäten

Ergänzen Sie nun die Webseite aus Aufgabe 2 um zwei eigenständige Sektionen. Diese sollen jeweils Überschriften der zweiten Ebene (<h2> besitzen.

Verwende in der ersten Sektion eine nicht nummerierte Liste ( <ul> ) mit entsprechenden Einträgen. Für die zweite Sektion sollten Sie eine kleine Tabelle mit drei Spalten, vier Zeilen und entsprechenden Kopf- und Fußzeilen ( <thead> und <tfoot> ) einfügen.

01. Ziffernblock: Tasten

Erstelle einen Ziffernblock, wie er auf einem Taschenrechner zu finden ist. (vgl. Num-pad auf der Tastatur)

Erstelle zuerst eine Div-box in der die anderen Elemente eingefügt werden. Gib dieser Div-box einen Klasse class="", um sie später addressieren zu können.

Nutze für die einzelnen Ziffern <button> Elemente. Um diese in Zeilen zu bringen, verwende den Zeilenumbruch <br>.

02. Ziffernblock: Interaktion

Erstelle ein Javascript Modul, welches du extern einbindest.

Erstelle darin die Klasse Calculator mit leerem Konstruktor.

Füge eine Methode onInputNumber(number) hinzu, welche den übergebenen Parameter in der Konsole ausgibt.

Füge eine Methode addNumpadListeners() hinzu und rufe diese im Konstruktor auf.

Setze in dieser Methode für jeden Button einen Listener document.addEventListener("eventName", function) hinzu der auf Klicks die Methode onInputNumber(number) aufruft.

03. Bildschirm

Erstelle eine <textarea> und füge dieser folgende Attribute hinzu:

  • class="bildschirm"
  • disabled
  • style="resize: none" (Morgen mehr dazu)
  • value="Hallo Welt"

Du solltest nun weder in die textarea schreiben können, noch sie skalieren können.

Greife nun per document.querySeletor() aus der Konsole auf die Textarea zu und ändere ihren Inhalt.

01. Ziffernblock: erweitert

Falls du den Ziffernblock manuell erstellt hast, ersetze dies nun mit einem programmatischen Ansatz. d.h. füge eine Methode zum Calculator hinzu die mit einer Schleife die Div-Box mit Buttos füllt. Verbinde dies sinnvoll mit dem Setzen der Listener auf diesen Elementen.
