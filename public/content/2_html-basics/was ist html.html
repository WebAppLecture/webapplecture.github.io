Was ist HTML?

HTML ist die Standard Markup Sprache mit der Webseiten geschrieben werden.

  • HTML steht für Hyper Text Markup Language
  • HTML beschreibt die Struktur von Webseiten
  • HTML Elemente sind die Bausteine, aus denen Webseiten bestehen
  • HTML Elemente werden durch Tags <tag-name> dargestellt
  • HTML Tags beschreiben Inhalte einer Webseite (Überschriften, Paragraphen, Tabellen, ...)
  • Browser zeigen HTML Tags nicht an, sondern benutzen sie um die Inhalte darzustellen
