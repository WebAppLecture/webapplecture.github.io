Einstieg in CSS

  • CSS (Cascading Style Sheets) erlaubt uns HTML Dokumente graphisch zu erweitern.
  • Style sheets bestehen aus Regeln die mit Selektoren HTML Elementen graphische Eigenschaften zuordnen: 
    selektor1, selektor2, ... {
    eigenschaft: wert;
    eigenschaft: wert;
}
  • Wo die Regeln definiert werden ist uns überlassen
    • Einbindung über eine externe Datei, üblicherweise im <head> des HTML Dokuments 
      <link rel="stylesheet" href="beispieldatei.css">
    • Direkt im HTML Dokument definiert 
      <style> ... </style>
    • Direkt im HTML Element definiert, hier definierte Regeln überstimmen andere Regeln! 
      <element style="...">
    • Über @import können wir weitere Stylesheets innerhalb eines Stylesheets einbinden, z.B. 
      @import url("/css/main.css");
  • Einige Eigenschaften vererben an Kindelemente, so dass es z.B. reicht die Schriftart nur einmal zu setzen
  • Es wird die spezifischste Regel angewendet
  • Später geladene Regeln überstimmen frühere (Dokumente werden von oben nach unten geladen)
