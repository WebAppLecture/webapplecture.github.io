Licht und Schatten

  • Mit CSS3 sind Schatten für Texte und Boxen möglich
  • Die Syntax für Boxen lautet 
    #eins { box-shadow: 5px 5px 0 0 #666; }
        #zwei { box-shadow: 0 0 5px 10px #F00 inset; }
  • Bei Text ist die Syntax sehr ähnlich z.B. 
    #drei { text-shadow: 0 0 5px red; }
        #vier { text-shadow: -3px 3px 2px #0F0; }
  • Die Syntax ist also immer so aufgebaut:
    • Versatz horizontal (pos. rechts, neg. links)
    • Versatz vertikal (pos. unten, neg. oben)
    • Breite (nur bei box-shadow)
    • Verschwommenheit (0 entspricht klar)
    • Farbe des Schattens
    • Richtung outset (Standard) bzw. inset
  • Schatten können auch überlagert werden, wodurch sehr komplexe Effekte erstellt werden können. 
    box-shadow: 
    0 0 5px 0 black inset,
    0 0 0px 10px #aaa inset,
    0 0 5px 12px black inset;
