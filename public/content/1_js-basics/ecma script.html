ECMA Script

ECMA Script bezeichnet den Sprachkern auf dem JavaScript und CoffeeScript basieren.
  • Version 1 erschien 1997, 2 und 3 in den Folgejahren
  • Version 4 wurde aufgrund von Sprachkomplexität verworfen. 5 erschien 2009, 5.1 2011
  • Juni 2015 erschien die Version 6 "Harmony" (auch ES6/ES2015), die umfassende Neuerungen brachte. Ab hier "modernes Javascript"
  • Ab 2015 Versionen im Jahrestakt, aktuell Version 11 (ES2020), Browser haben aber auch schon teilweise Support für ES2021 Features
  • ES6 ist mittlerweile fast voll unterstützt
  • Vergleiche Prozentsatz an unterstüzter ES6 Funktionalität Februar 2019 / September 2020
    • Chrome V8 basierte Browser: 98% / 98%
    • SpiderMoney basierte Browser (Firefox): 98% / 98%
    • Chakra basierte Browser (MS Edge): 96% / 98% (jetzt Chrome basiert)
    • JavascriptCore basierte Browser (Safari, IOS): 99% / 100% (IOS 99%)
    • Node.js > V10.9: 98% / 98%
    • IE11: 11% / 11%
  • Es gibt Programme, die es erlauben ES 6 Code in ES 5 zu übersetzen (Babel, Traceur; Implementierung liegt hier bei 50-70%)
ES 6 enthält viele Neuerungen, die für uns interessantesten Features sind
  • Verwendung von Klassen und Modulen
  • Variablen-Deklaration mit const und let
  • DOM Element Selektion mit document.querySelector
  • Fat Arrow Funktionen () =>
  • Datentypen Map() und Set()
  • Spread operator ...
  • Destructuring let { a, b } = { a: 1, b: 2 }
