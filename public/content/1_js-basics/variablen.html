Variablen deklarieren

Javascript ist dynamisch Typisiert.
  • Keine Deklaration des Variablentyps wie z.B in Java int i = 0 oder String s = "test"
  • Die Typen werden dynamisch während der Laufzeit bestimmt und können sich auch ändern! let i = 0; i = "test";
  • Deklaration einer Variablen erfolgt mit ihrer ersten Aufzählung (let a)
  • Es gibt auch eine streng typisierte Sprachvariante Typescript, die deswegen besonders bei großen Projekten sehr beliebt ist.
Mit ES 6 erhiel Javascript zwei neue Bezeichner für Variablen: const und let
  • Der alte var Bezeichner sollte nicht mehr benutzt werden!
  • let verhält sich wie eine normale Variable, ohne var's Eigenheiten
  • const bezeichnet eine nicht veränderbare Konstante.
    • Konvention: Verwende GROSSBUCHSTABEN für Konstanten. z.B. const BG_COLOR = "#fff"
    • Verhindert nicht das Verändern von Eigenschaften der Variable! (siehe Beispiel)
  • Variablendeklarationen lassen sich kurz fassen: let a = 1, b = 2; enspricht let a = 1; let b = 2;
