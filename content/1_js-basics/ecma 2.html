ECMA 2016+

Einige der wichtigsten Änderungen der letzten Versionen
  • ES2016
    • Potenzoperator **
    • Array.prototype.includes
  • ES2017
    • Object.values und Object.entries
    • Asynchrone Functionen mit asyncawait
  • ES2018
    • Rest/Spread operator für Objektea = {...b}
    • Promise.prototype.finally
    • Benannte fangende Gruppen in Regex
    • Asynchrone Iteratoren
  • ES2019
    • Maps in Objekte konvertieren Object.fromEntries
    • String trimming
    • Array.prototype.{flat, flatMap}
  • ES2020
    • String.prototype.matchAll (endlich!)
    • BigInt für größere Integer
    • globalThis
    • Optional chaining operator undefined?.property === undefined
    • Nullish coalescing operator ??
  • ES2021
    • String.prototype.replaceAll
    • Promise.any
    • Logische Zuweisung ||=, &&=, ??=
    • Nummeric Separators für Lesbarkeit 1000000 === 1_000_000
Support für ES2016+ (Stand September 2020):
  • Chrome (> CH 85): 100%
  • Firefox (> FF79): 100%
  • MS Edge (alt, v18): 33%
  • Chromium Basiert (Edge, Opera): 87%
  • Safari (> SF 13.1, IOS > SF 13.4): 79%
  • Node.js > V14.0: 87%
  • IE11: 1%
