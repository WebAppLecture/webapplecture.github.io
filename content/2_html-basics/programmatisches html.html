Programmatisches HTML

HTML kann nicht nur direkt, sondern auch per Javascript generiert werden.
  • So kann man während der Laufzeit Inhalte verändern/hinzufügen/löschen
  • Wichtige Befehle:
    • document.createElement(X) um ein Element <X> zu erstellen.
    • .appendChild() um es einem anderen Element unterzuordnen.
    • .innerHTML bzw .innerText um den Inhalt eines Elements direkt zu verändern.
    • .classList gibt uns Zugriff auf die gesetzten Klassen.
    • .dataset gibt uns Zugriff auf alle data- Attribute.
    • .stlye gibt uns Zugriff auf das Styling.
    • .setAttribute(name, value) setzt eine eigenschaft auf einem Element.
    • .remove() löscht das Element.
    • .children gibt uns eine Liste der Kindelemente. Achtung: ist kein normales Javascript Array! Mit Array.from() konvertieren.
  • Wichtig: Objekte hinzuzufügen (.appendChild()) ist effizienter als innerHTML zu verändern
