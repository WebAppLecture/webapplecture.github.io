Interaktion über Ereignisse

  • Ereignisse (Events) sind Aktionen die von JavaScript detektiert werden können
  • Jedes HTML Element hat einige bestimmte Ereignisse, die wir nutzen können
  • Mit .addEventlistener(event, function) melden wir Funktionen an, welchen beim Eintreten des Events aufgerufen werden.
  • Mit .removeEventlistener(event, function) entfernen wir sie wieder.
  • Beispiele für Ereignisse:
    • Ein Mausklick (click)
      • Start: mousedown
      • Ende: mouseup
    • Ein Tastendruck (keypress)
      • Start: keydown
      • Ende: keyup
    • Absenden des Formulars (submit)
    • Eingabe / Änderung von Formelement (change)
    • Betreten der Seite (load)
    • Mit der Maus über ein Element fahren (mouseover)
  • JavaScript ist komplett ereignisgesteuert
