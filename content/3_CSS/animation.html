Animationen ohne JS

  • Effekte wie eben können wir natürlich auch ohne CSS3 in JavaScript erledigen (per CSS ist schneller!)
  • Wie sieht es aber nun aus wenn wir nicht nur Übergänge handhaben wollen sondern echte Animationen?
  • Selbst hier können wir auf JavaScript verzichten - CSS3 Animationen!
    • Definiert über @keyframes
    • Syntax je Frame: zeitpunk { eigenschaft: wert}
    • Zeitpunk in % von 0 bis 100
    • Alternativ: from = 0%, to = 100%
    • animation-name Referenz auf den Keyframenamen.
    • animation-duration Wie lang läuft die Animation, in Sekunden.
    • animation-iteration-count Wie oft läuft sie durch.
    • animation-timing-function Wie wird zwischen den Frames interpoliert.
    • animation-delay Wie lange wartet die Animation, bis sie anfängt.
    • animation-direction In welcher Richtung werden die Keyframes durchlaufen.
    • animation-play-state Zum Pausieren der Animation.
    • animation-fill-mode Zustand vor und nach der Animation.
    • Auch als Kurzschreibweise: animation: name duration timing delay count direction fill state;
