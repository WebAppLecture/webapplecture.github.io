Die Größe festlegen

  • width bzw. height stellen eine feste Größe ein
  • Wir können mit min-width bzw. max-width minimale bzw. maximale Größen einstellen
  • Nicht nur
  • Über margin kann man Abstand zu anderen Elementen einstellen
  • border setzt den Rahmen (Breite, Stil, Farbe)
  • padding setzt den inneren Abstand (Breite)
  • Das CSS Box-Model bestimmt die Größe von Elementen auf der Seite.
  • Wie groß das Element dann genau ist berechnet sich auf Basis des gewählten Box-Models:
    • border-box: width/height legen die Gesamtgröße des Elements fest, border und padding verkleinern die Größe des Inhalts
    • content-box: Die Größe des Contents wird gesetzt, padding und border addieren sich dazu (Standard des Browsers!)
