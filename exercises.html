Erste Schritte

Öffne eine beliebige Seite im Browser und öffne die Entwicklertools. (Normalerweise F12)

Navigiere zur Konsole und erstelle dir zwei variablen a=2 und b=3.5.

  • Probiere nun mal die Grundrechenarten aus.
  • Was passiert wenn du eine der Variablen durch 0 teilst?
  • a + "a", wie verhält sich Javascript wenn man verschieden Typen verrechnet?
  • (3 + "" + 2) * 1 Implizierte Casts ermöglichen uns Datentypen zu erzwingen.

01. Das Grundgerüst

Erstelle die Datei "number.html", und füge in diese einen <script> Tag ein.

Erstelle darin die Klasse Number, diese erhält als einzigen Konstruktorparameter den Initialwert ihrer Variablen value, und soll, falls dieser nicht gegeben ist, diesen auf 0 setzen.

Öffne die Datei im Browser, öffne mit F12 die Entwicklertools und initialisiere deine Klasse: a = new Number(3)

Stelle sicher, dass eine Initialisierung ohne Parameter zum gewollten Wert 0 führt.

02.Grundrechenarten

Füge nun Funktionen für die Grundrechenarten zu der Klasse Number hinzu: add(), subtract(), multiply(), divide()

Schau dir an was passiert, wenn du in Javascript durch 0 teilst. In unsere Klasse wollen wir dies unterbinden, und bei einer Division durch 0 den Wert nicht verändern.

Kontrolliere wieder in der Konsole deinen Fortschritt. a = new Number(); a.add(4); sollte dazu führen dass a.value 4 zurückgibt.

03. Chaining

Wir wollen nun Ausdrücke wie a.add(4).multiply(3) ermöglichen, dies schaffen wir mithilfe von Chaining.

Chaining erfordert einfach nur, dass jede (chainable) Funktion einer Klasse die aktuelle Instanz dieser zurückgibt. Dies erreichen wir indem wir this zurückgeben.

Spiel ein wenig mit den neuen Funktionalitäten in der Konsole.

04. Potenz

Füge nun eine Funktion pow() hinzu, die uns erlaubt zu potenzieren.

Verwende intern nicht die in Javascript eingebaute Math.pow() Methode, du kannst aber mithilfe dieser in der Konsole dein Ergebniss kontrollieren.

Eine For-Schleife könnte hier hilfreich sein. Beachte, dass diese Lösung keine negativen Potenzen (Wurzeln) vorsieht. Überprüfe deswegen ob die übergebene Potenz positiv ist.

05. Fakultät

Schreibe eine Funktion faculty(), die ein Argument die Fakultät n! des aktuellen Wertes unserer Number berechnet.

Die Formel hierzu lautet: n! = n ⋅ (n - 1) ⋅ (n - 2) ⋅ ... ⋅ 1

Die Fakultät ist nur für Ganzzahlen definiert, überprüfe deswegen vorher ob deine aktuelle Zahl dies erfüllt. Dies ist mit n % 1 === 0 überprüfbar, der Modulo-Operator % berechnet hier den Rest der Division durch 1, der bei Ganzzahlen immer 0 ist.

Falls dies nicht erfüllt ist, ignoriere die Anweisung und ändere den Wert nicht.

06. Auslagerung

Erstelle die Datei number.js und velagere die Klasse Number dort hin.

Füge ein export statement zur Datei hinzu und importiere die Klasse in number.html. <script type="module"> import Number from "number.js"; </script>

Stelle sicher, dass alle Funktionalität fortbesteht.
