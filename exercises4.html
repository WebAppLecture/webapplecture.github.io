01. Audio

Lade dir aus dem Internet eine beliebige Audio-Datei herunter, die von dem benutzten Browser unterstützt wird.

Erstelle in einer neuen html Seite ein audio Element und füge diesem die Datei als quelle hinzu.

Schalte mit dem Attribut controls die eingebauten Steuerelemente ein.

Erstelle "Player.js" und erstelle darin die Klasse Player. Player...

  • ...erhält eine Referenz auf ein audio element im Konstruktor.
  • ...hat Methoden play, pause und setVolume.
  • ...setzt Listener auf das übergebene audio Element um diese Methoden aufzurufen, wenn mit dem audio Element interagiert wird.

Dynamischer Graph

Erstelle eine svg Element.

Erstelle im svg ein Koordinatensystem mit zwei Achsen. Zeichne diese einfach mit polylines.

Erstelle einen textinput, einen value slider (Wertebereich selbst festlegen), einen color input und einen Button "erstellen". Erstelle eine Klasse die die Interaktion mit diesen Elementen kontrolliert:

  • Wenn auf den Button geklickt wird soll ein neues svg Rechteck erstellt werden.
    • Das Rechteck ist so hoch wie der wert des sliders (relativ zur svg Höhe und eurem Wertebereich).
    • Das Rechteck hat die Farbe die im color input selektiert ist.
    • Unter dem Rechteck ist der Inhalt des Textinputs angezeigt. (svg text tag)
    • Die Breite des Rechtecks richtet sich nach der Menge der vorhandenen Rechtecke, mehr Rechtecke sind schmaler.

Füge einen Weiteren Button "löschen" hinzu.

Wenn auf ein Rechteck geklickt wird, soll dieses selektiert werden. d.h.

  • Dem Element wird die css-Klasse selected-graph gegeben.
  • Diese Klasse sollte so gestylt sein, dass der Graph hervorgehoben ist.

Klick auf den "löschen" Button soll alle hervorgehobenen Buttons aus dem svg entfernen.
