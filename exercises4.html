01. Audio

Lade dir aus dem Internet eine beliebige Audio-Datei herunter, die von dem benutzten Browser unterstützt wird.

Erstelle in einer neuen html Seite ein audio Element und füge diesem die Datei als quelle hinzu.

Schalte mit dem Attribut controls die eingebauten Steuerelemente ein.

Erstelle "Player.js" und erstelle darin die Klasse Player. Player...

  • ...erhält eine Referenz auf ein audio element im Konstruktor.
  • ...hat Methoden play, pause und setVolume.
  • ...setzt Listener auf das übergebene audio Element um diese Methoden aufzurufen, wenn mit dem audio Element interagiert wird.
